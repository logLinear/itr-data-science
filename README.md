# itr-data-science
All assigned tasks for a 4-week internship (16-05-2026 to 16-06-2026) in 'Data Science Using Python' by Soft Nexis Technologies


- ## Task 1: Data Cleaning & Preprocessing
Successfully performed data ingestion, deduplication, column management, missing value handling, data type correction, and format standardisation were performed on Customer dataset with 100,000 entries. 
It has the following fields: Index, Customer Id, First Name, Last Name, Company, City, Country, Phone 1, Phone 2, Email, Subscription Date, Website.

**Steps to run this project:**
1. Clone the repository
2. Place `customers-100000.csv` in the task_1/ folder
3. Run `pip install -r requirements.txt`
4. Run `python task_1/main.py`
