# itr-data-science
All assigned tasks for a 4-week internship (16-05-2026 to 16-06-2026) in 'Data Science Using Python' by Soft Nexis Technologies


- ## Task 1: Data Cleaning & Preprocessing
Successfully performed data ingestion, deduplication, column management, missing value handling, data type correction, and format standardisation on Customer dataset with 100,000 entries. 
Customer dataset has the following fields: Index, Customer Id, First Name, Last Name, Company, City, Country, Phone 1, Phone 2, Email, Subscription Date, Website.

**Steps to run this project:**
1. Clone the repository
3. Run `pip install -r requirements.txt`
4. Run `python task_1/main.py`


- ## Task 2: Exploratory Data Analysis (EDA)
Performed exploratory data analysis on S&P 500 stock price data with ~619,000 entries across 500+ stocks. Analysis covers distribution analysis, correlation & relationships, outlier detection, and advanced visualizations including trend lines, heatmaps, and pairplots. S&P 500 stock prices dataset has the following fields: Date, Open, High, Low, Close, Volume, Name.

**Steps to run this project:**
1. Clone the repository
2. Open `task_2/main.ipynb` in Jupyter Notebook, Google Colab, or VSCode
3. If using VSCode, then run `pip install -r requirements.txt`
4. If using Jupyter Notebook or Google Colab, no need for installment.
6. Run all cells top to bottom
