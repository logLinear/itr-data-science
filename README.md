- ## Task 1: Data Cleaning & Preprocessing
The following steps were performed on Customer dataset with 100,000 entries. It has the following fields:
Index, Customer Id, First Name, Last Name, Company, City, Country, Phone 1, Phone 2, Email, Subscription Date, Website

1. **Data Ingestion** — Load dataset with sanity checks.
2. **Deduplication** — Remove exact and partial duplicates.
3. **Column Management** — Drop irrelevant columns, rename, and reorder.
4. **Missing Value Handling** — Diagnose and impute missing values.
5. **Data Type Correction** — Fix datetime and phone number formats.
6. **Format Standardisation** — Normalise text, whitespace, and casing.
